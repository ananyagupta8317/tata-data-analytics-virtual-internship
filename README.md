# Tata Data Analytics Virtual Internship (Forage)

## Overview
The project simulates a real-world retail analytics scenario where data-driven insights are provided to senior stakeholders (CEO and CMO).

## Tasks Completed
- Framed business questions from CEO and CMO perspectives
- Solved analytical MCQ-based assessments
- Cleaned retail transaction data using Excel
- Built Power BI visualizations for revenue, customers, and demand analysis
- Presented insights through a recorded business presentation

## Tools Used
- Excel (Data Cleaning, Conditional Logic)
- Power BI (Data Visualization, Business Analysis)
